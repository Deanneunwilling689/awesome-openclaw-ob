# TrustMRR 追踪的 117 家 OpenClaw 创业公司

## 一句话总结

TrustMRR（创业公司收入验证平台）追踪到 117 家基于 OpenClaw 的创业公司，集体 MRR 超过 $100K，其中 SimpleClaw 上线两周即达 $30K MRR，但多家 wrapper 公司已挂牌出售。

## 核心数据

| 指标 | 数据 |
|------|------|
| 追踪创业公司总数 | **117 家**（TrustMRR 验证） |
| 其中 wrapper 类 | 104+ 家 |
| 集体 MRR | **$100K+**（已验证） |
| SimpleClaw MRR | 上线 2 周达 $30K，30 天 $41K+ |
| SimpleClaw 模式 | 一键部署 OpenClaw |
| OpenClawd 定位 | 托管平台，类比 "Vercel 之于 Next.js" |
| ClawBox 模式 | SaaS 月订阅制，专属托管服务器 |
| Atomic Bot | 跨平台客户端，100+ 连接器 |
| EveryClaw.ai | 零配置部署，无需 DevOps |

## 分析

围绕 [[OpenClaw 商业模式|OpenClaw]] 的创业生态呈现出典型的"开源项目 wrapper 淘金热"模式。117 家公司中超过 104 家是 wrapper 类——本质上是给 OpenClaw 加一层部署/管理界面然后收费。SimpleClaw 两周 $30K MRR 的案例看似亮眼，但需要冷静看待：

**壁垒极低**是核心风险。这些 wrapper 公司的技术护城河几乎为零——一旦 [[OpenClaw Foundation 治理|OpenClaw 基金会]]或某个大厂推出官方云版本，90% 的 wrapper 可能失去生存空间。事实上，多家 wrapper 创业公司已同时在 TrustMRR 挂牌出售，创始人套现离场的信号明显。

更有价值的是 OpenClawd 和 ClawBox 这类有差异化定位的公司——前者声明独立于 [[OpenAI 为什么收编 OpenClaw|OpenAI]] 和基金会，后者提供多渠道企业服务。

## 关键洞察

117 家创业公司的集体 $100K+ MRR 本身不算大数字，但它证明了 [[OpenClaw 生态系统商业机会|OpenClaw 生态]]的商业需求是真实的——有人愿意付费解决部署和运维的痛点。真正的投资机会不在 wrapper 层，而在[[OpenClaw 投资机会因素|基础设施层]]：安全（[[OpenClaw 投资风险因素|Snyk]] 的 Agent Trust Hub）、路由（[[OpenRouter 融资与商业|OpenRouter]] 的 $5 亿估值）、监控。这些是 OpenClaw 生态中有持续壁垒的赛道。wrapper 创业公司的快速涌现和快速出售，恰恰说明了 **"卖铲子"比"淘金"更值得关注**。
