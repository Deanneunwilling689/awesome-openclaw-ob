# OpenRouter 融资与商业

OpenRouter 是 LLM 路由服务，为 [[OpenClaw 是什么|OpenClaw]] 生态提供关键基础设施。它支持 [[模型无关架构]]，让开发者一个 API 接入多种模型。

## 融资详情

| 轮次 | 金额 | 领投方 | 时间 |
|------|------|--------|------|
| Seed | $12.5M | a16z (Sequoia 跟投) | 2025年初 |
| Series A | $28M-$47.6M | Menlo Ventures (Sequoia 跟投) | 2025年6月 |
| **合计** | **官宣 $40M（Tracxn 记录 $60.1M）** | | |

> Series A 金额存在来源差异：Sacra 报 $28M，Tracxn 报 $47.6M，官方新闻稿合计 $40M。

## 关键指标

- **估值**：约 $5 亿（Post-money）
- **创始人**：Alex Atallah（OpenSea 联合创始人）和 Louis Vichy（2023年创立）
- **增长**：年化推理支出从 $10M（2024.10）增至 $100M+（2025.5）
- **用户**：100 万+ 开发者
- **商业模式**：所有推理支出收取 **5% 佣金**，接入 400+ LLM 模型，包括 Claude、GPT、DeepSeek 等

## 投资人评价

a16z Anjney Midha：
> "AI stacks are fragmenting. OpenRouter is unifying them."
