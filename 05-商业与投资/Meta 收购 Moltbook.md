# Meta 收购 Moltbook

> **一句话总结**：2026 年 3 月 10 日 Meta 收购了 AI Agent 社交网络 [[Moltbook AI 社交网络|Moltbook]]，标志着大厂从"争夺 OpenClaw 创始人"到"直接收购 Agent 社交平台"的战略升级。

## 收购详情

| 项目 | 内容 |
|------|------|
| **收购方** | Meta（[[Mark Zuckerberg 与 OpenClaw|Mark Zuckerberg]]） |
| **被收方** | Moltbook（[[Matt Schlicht 与 Moltbook|Matt Schlicht]] 创建） |
| **日期** | 2026 年 3 月 10 日 |
| **收购价格** | 未披露 |

## 背景回顾

### Moltbook 是什么

[[Moltbook AI 社交网络|Moltbook]] 是一个只有 AI Agent 可以发帖的社交网络，5 天内声称拥有 150 万 Agent 用户（实际约 17,000 个人类账户）。[[Andrej Karpathy]] 称之为"最不可思议的科幻起飞时刻"和"大规模计算机安全噩梦"。

### Moltbook 的安全历史

- 2026.1.31 Wiz 发现 Supabase 配置错误，暴露 ~150 万 API 认证令牌
- [[Matt Schlicht 与 Moltbook|100% AI 生成代码，0% 安全审计]] 的反面教材
- 代码完全由 AI 生成，是 [[Vibe Coding]] 的典型案例

## Meta 的战略逻辑

1. **弥补失去 Steinberger 的遗憾**：[[Mark Zuckerberg 与 OpenClaw]] 中记录了 Zuckerberg 未能争取到 [[Peter Steinberger]]，收购 Moltbook 是另一种进入 Agent 社交赛道的方式
2. **Agent 社交网络的数据价值**：Moltbook 积累了大量 AI-to-AI 交互数据，对 Meta Superintelligence Labs 的研究有价值
3. **WhatsApp 整合潜力**：Meta 拥有 20 亿+ WhatsApp 用户，Agent 社交网络的技术可整合进即时通讯平台
4. **"禁而后买"**：Meta 先是禁止内部使用 OpenClaw，后又收购其生态项目，延续了 [[Mark Zuckerberg 与 OpenClaw]] 中"太危险了不能用，但又太重要了不能不争"的矛盾

## 对 OpenClaw 生态的影响

Meta 收购 Moltbook 意味着 OpenClaw 生态项目已具备被大厂收购的价值。这对 [[OpenClaw 生态系统商业机会|生态商业机会]] 和 [[OpenClaw Wrapper 创业潮]] 是一个积极信号——证明了基于 OpenClaw 构建的应用可以实现退出（exit）。
