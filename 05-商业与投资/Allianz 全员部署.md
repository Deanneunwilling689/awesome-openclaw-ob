# Allianz 全员部署

Allianz（安联保险集团）是全球最大的保险和资产管理公司之一，其在 AI Agent 领域实施了全员级别的部署方案，是企业级 AI 采用的标杆案例。

## 背景

在 [[Anthropic Agentic Coding 趋势报告]] 中，Allianz 的全员 AI 部署与 Zapier（89% AI 采用率）、TELUS（13,000+ 定制方案）并列为企业 AI 规模化应用的典型案例。

## 意义

Allianz 的案例证明 AI Agent 不再局限于技术团队——保险行业的理赔、核保、客服等业务线均可受益于 Agent 自动化。这与 [[2026 Agent 元年]] 中"从实验到部署"的行业趋势一致。
