# 游戏引擎插件集成

这是 [[OpenClaw 是什么|OpenClaw]] 生态中一个典型的跨界应用案例。

## 概况

- **作者**：Tom Jaejoon Lee
- **支持引擎**：Unity、Godot、Unreal
- **工具数量**：170+
- **下载量**：1,250+
- **许可证**：Apache 2.0

## 功能

- 创建对象、修改场景
- 模拟输入
- 运行 Play Mode
- 全部通过聊天界面操作（利用 [[Tool Use 机制]]）

## 意义

游戏引擎插件展示了 AI Agent 超越传统软件开发场景的能力，体现了 [[OpenClaw 是什么|OpenClaw]] 作为通用 Agent 框架的灵活性。
