# 社区项目速览

以下是 [[OpenClaw 是什么|OpenClaw]] 社区的更多开源项目，展示了 [[Tool Use 机制]] 在各领域的应用广度。

## 项目列表

| 项目 | 创作者 | 描述 |
|------|--------|------|
| 游戏引擎插件 | Tom Jaejoon Lee | Unity/Godot/Unreal 编辑器 AI 控制，170+ 工具，1250+ 下载 |
| Home Assistant 集成 | @ngutman | 树莓派上部署，自然语言控制智能家居 |
| 3D 打印机控制 | @tobiasbischoff | Bambu 3D 打印机状态、作业、摄像头管理 |
| 维也纳公交查询 | @hjanuschka | 实时出发时间、故障、电梯状态 |
| 健康助理 | @AS | Oura Ring 数据 + 日历 + 健身房排期整合 |
| 天文设备市场 | Community | star-swap.com 天文设备交易平台 |

## 趋势观察

这些项目覆盖了游戏开发、智能家居、硬件控制、公共交通、健康管理、电商等领域，说明 [[Agentic Coding]] 的应用范围远超"写代码"本身。
