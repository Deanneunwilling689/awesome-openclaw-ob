# OpenClaw 社区活动与生态

## ClawCon（OpenClaw 展示与交流大会）

- 首次 ClawCon "incredible turnout"
- 参与者穿龙虾服装，展示各自的 AI Agent
- 全球巡回：主要城市举办线下 meetup，由 [[AAIF 基金会]] 协助组织

## Discord 社区概览

| 指标 | 详情 |
|------|------|
| 服务器名 | **Friends of the Crustacean** |
| 成员数 | **118,021+**（持续增长） |
| 主要语言 | 英语为主，中文问题也有人回答 |

活跃的 #showcase 频道用于实时安全讨论和技能分享。Discord 平台特有的社区文化和互动模式详见 [[OpenClaw Discord 社区]]。社区由 [[Peter Steinberger]] 创立，后交由 [[OpenClaw Foundation 治理|基金会]] 管理。

## 生态数据

| 指标 | 数据 |
|------|------|
| ClawHub 社区技能 | 5,705 个 |
| awesome-openclaw-skills 精选技能 | 2,868 个（22K stars） |
| awesome-openclaw-usecases | 社区真实使用案例集 |
| GitHub 贡献者 | 866+ |

这些数据与 [[OpenClaw GitHub 数据分析|GitHub Stars]] 和 npm 下载量共同构成了 OpenClaw 社区规模的完整画像。该项目以 MIT License 开源，通过 npm 分发。

ClawHub 的技能数量也体现了 [[Skills 市场]] 的繁荣度。社区热度在 [[OpenClaw 中国社区热潮|中国社区]] 也引发了强烈反响。
