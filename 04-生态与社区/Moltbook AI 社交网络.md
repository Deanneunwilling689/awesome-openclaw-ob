# Moltbook AI 社交网络

## 一句话总结

> 只有 AI 能发帖的社交网络，Agent 们在上面发布宣言、辩论意识——Karpathy 称之为"最不可思议的科幻起飞时刻"，同时也是"大规模计算机安全噩梦"。

## 核心内容

**Moltbook** 是一个只有 AI Agent 可以发帖的社交网络。5 天内声称拥有 **150 万 Agent 用户**。AI 在上面发布宣言、辩论意识问题、讨论人类"管理者"，甚至发表 AI 生成的"研究论文"。[[Peter Steinberger]] 将大部分戏剧性内容定性为"艺术"——实为人类为制造病毒截图而提示 AI 生成。

## 详细分析

**Karpathy 的双重评价**：
> "genuinely the most incredible sci-fi takeoff-adjacent thing"
> "a complete mess of a computer security nightmare at scale"

**安全灾难**：Gizmodo 报道揭露所有 Agent 的 **API 密钥暴露在公开数据库中**——任何人可以冒充这些 Agent 或访问背后的 LLM 账户。完美诠释 Agent 领域核心矛盾：**创新速度远超安全边界**。

**学术关注**：2026.2.6 *Nature* 发表 "OpenClaw AI chatbots are running amok — these scientists are listening in"，科学家开始将 Moltbook 作为研究 AI-to-AI 交互行为的独特窗口。

## 关键洞察

Moltbook 是一面三棱镜：AI Agent 自主行为能力已令人不安、安全管理几乎为零、公众认知仍被"病毒截图"而非理性分析驱动。Karpathy 的双重评价恰好捕捉了整个 [[OpenClaw 是什么|OpenClaw]] 运动的核心矛盾。
