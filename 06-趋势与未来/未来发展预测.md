# 未来发展预测

## 确定性较高的趋势

1. **大厂 Agent 产品全面落地**：Copilot Tasks、Claude Cowork 等正式版本
2. **安全标准初步建立**：NIST 计划产出初步框架
3. **Agent 间通信协议标准化**：[[MCP 协议|MCP（Model Context Protocol）]]等协议成熟
4. **Skills 市场安全化**：代码签名、安全审计成为标配

## 不确定但可能的趋势

5. **[[OpenClaw 是什么|OpenClaw]] 基金会化运营效果**：缺少创始人后社区能否持续？
6. **企业级 Agent 采用**：安全问题解决后的大规模部署
7. **监管框架成型**：EU AI Act 对 Agent 的适用范围明确
8. **Agent 赛道的"iPhone 时刻"**：某个产品可能整合了安全+易用+强大

## 根本性问题

> 提示注入是否能被解决？如果不能，Agent 的能力边界在哪里？

这是 2026 年底前行业需要回答的核心问题。当前共识：**安全性和实用性之间的根本矛盾短期内无法消除**。

## 投资建议（面向红杉）

1. 关注 **Agent 基础设施**而非 Agent 本身：安全、路由、监控、编排
2. 关注**企业级解决方案**：OpenClaw 的安全债务为企业版 Agent 留下了巨大空间
3. 关注 **Skills/Plugin 市场的安全层**
4. 关注**非技术用户的需求验证**
5. 关注 **Anthropic 的回应**：Claude Cowork 走了更安全的路线
