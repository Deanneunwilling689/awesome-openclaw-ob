# Replit Agent 分析

## 基本信息

- **类别**：AI 编码代理
- **定位**："[[Vibe Coding|vibe coding]]" 快速原型，编码民主化
- **目标用户**：非技术人员 → 软件构建者（"十亿开发者"愿景）

## 市场表现

- **$30 亿估值**（2025.9，$2.5 亿融资）
- 从 $280 万飙升至 **$1.5 亿年化收入**（<1 年），增长 50 倍+
- **75 万+** 企业用户

## 旗舰案例

**Rokt**：24 小时内构建 135 个内部应用（700+ 员工参与）

## Agent 3 能力

- 自主性提升 **10 倍**
- 可连续工作 **200 分钟**

## 与 OpenClaw 的差异

两者的"降低门槛"方向不同，但都依赖 LLM 驱动：
- **Replit** 降低的是**编码门槛**（让非技术人员也能构建应用），属于 [[Agentic Coding]] 范畴
- **[[OpenClaw 是什么|OpenClaw]]** 降低的是**AI 代理的使用门槛**（通过聊天界面自动化日常生活）
