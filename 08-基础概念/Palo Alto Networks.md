# Palo Alto Networks

Palo Alto Networks 是全球领先的网络安全公司（纳斯达克：PANW），在 AI Agent 安全领域积极发声。

## 对 OpenClaw 的安全评估

Palo Alto Networks 将 [[OpenClaw 是什么|OpenClaw]] 称为 **"2026 年最大的潜在内部威胁"**，指出当员工在企业环境中使用未经授权的 AI Agent 时，Agent 对敏感数据的自主访问权限构成严重安全风险。

这一评估呼应了 [[Shadow AI 现象]] 中的核心问题——29% 的企业员工在未经 IT 部门批准的情况下使用 AI Agent 工具。

## 在 AI 安全生态中的角色

作为传统网络安全巨头，Palo Alto Networks 正在将 AI Agent 安全纳入其产品线，为企业提供 Agent 流量监控、数据泄露防护和合规审计等能力。这代表了传统安全厂商向 [[AI Agent 安全现状|AI Agent 安全]] 领域的战略延伸。
