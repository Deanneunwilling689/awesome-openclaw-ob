# 非技术用户真实案例

以下案例展示了 [[编程民主化]] 的实际效果。

## 案例 1：Ken Yeung 的 ClawBeat.co

- **背景**：自称 "largely non-technical"，上次写代码是做 HTML/CSS
- **成果**：构建了完整的内容聚合平台，带 AI 摘要、YouTube 嵌入、ArXiv 论文追踪
- **方法**：ChatGPT 创建编辑规则 -> Google AI Studio 生成前端 -> Gemini 逐步开发 -> GitHub + Vercel 部署
- **评价**："My first true production repository"

## 案例 2：@coard 的 iOS 应用

- 通过 **Telegram 聊天**完成了完整的 iOS 应用（地图 + 语音录制）
- 部署到 TestFlight
- **全程没有打开 Xcode、终端或任何 IDE**

## 案例 3：超市自动购物

- @marchattonhere 通过 OpenClaw 浏览器控制
- 计划餐食 -> 选择食材 -> 预约 Tesco 配送
- 不需要任何 API 知识，纯浏览器操作

## 案例 4：帮助新手提交首个 PR

Lex Fridman 播客提到 OpenClaw 的 Agent 帮助不会编程的人完成了他们人生中第一次 GitHub Pull Request。

## 案例 5：律师 Emma 的效率革命

- 42 岁，芝加哥家庭法律师
- 用 OpenClaw 管理健康追踪和工作效率
- 每月仅 **$12**，无需任何编程知识

## 案例 6：My Legal Academy 律所自动化

- 为律所设计了 20 种自动化方案
- "多数律所因接案流程漏洞丢失 30-50% 潜在客户"
- Agent 自动跟进客户咨询、安排会议、生成文件模板

## 案例 7：教师评测

Tech & Learning 评测团队发现 OpenClaw 能查找住宿、总结课件、生成课程计划。

## 案例 8：全科医生的医疗应用

- 两三天内创建多个医疗辅助应用（患者随访、处方提醒、症状分类）
- 全程通过聊天界面完成，未写一行代码
